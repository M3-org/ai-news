# ElizaOS Development Report - January 6, 2026

## Technical Issues and Bug Fixes

### Discord Plugin Server ID Issues
- Error "No server ID found 10" traced to version compatibility issues between ElizaOS 1.7.0 and plugin-discord 1.3.3
- Root cause identified as transition from `serverId` to `messageServerId` in codebase
- Fix branch (odi-17) created with corrections for bootstrap's actions/providers
- PR #6333 submitted to address these issues

### Database Migration Errors
- Solution provided for destructive migration errors when running `elizaos start` a second time
- Options include setting `ELIZA_ALLOW_DESTRUCTIVE_MIGRATIONS=true` for local development or using `elizaos dev` command

### Performance Issues
- Turbo build excessive memory consumption reported, reaching 21GB or more (Issue #6332)
- Model not found error with agent ID when using API endpoints identified (Issue #6331)
