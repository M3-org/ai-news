# elizaOS Discord - 2025-06-13

## Overall Discussion Highlights

### ElizaOS Platform Status
- **ElizaOS V2 Release**: Version 1.0.8 is officially out of beta with significant improvements over V1, making development easier
- **Version 1.0.9 Announcement**: Released during the day to address several bugs from 1.0.8
- **Plugin Ecosystem**: The team highlighted they have 180+ plugins available, though not all are migrated to V2 yet
- **Database Migration Issues**: Several users reported problems with database migrations in v1.0.8, particularly with the error "Failed query: CREATE SCHEMA IF NOT EXISTS $user"

### Spartan Development
- **Architecture**: Spartan is a separate stack from ElizaOS with different architecture
- **Product Release**: First product focused on DeFi tools is imminent
- **Development Approach**: Team plans to release simple tools first and iterate based on community feedback

### Communication Challenges
- **Twitter/X Account Suspension**: Major community concern about the project's suspended Twitter/X accounts
- **Impact on Visibility**: Users expressed concern about the effect on price action and project visibility
- **Alternative Channels**: Team is working on expanding communication across multiple platforms as backup

### Technical Issues
- **Action Data Passing**: Users identified issues with actions overwriting each other's data (particularly WEB_SEARCH and REPLY)
- **Plugin Functionality**: Several plugins including Twitter and Telegram reported as having issues
- **Agent Persistence**: Questions about agent IDs changing between restarts and memory persistence across worlds
