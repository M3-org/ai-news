# elizaOS Development Discord - 2025-03-19

## Overall Discussion Highlights

### Eliza v2 Beta Status
- The v2 beta is currently experiencing various installation and setup issues
- Not considered stable enough for production use yet
- Development team is actively collecting feedback on bugs and issues
- The `v2-develop` branch is the latest version users should try

### Technical Discussions
- **Model Performance**: deepseek-r1:14b works well with Eliza due to its instruction-following capabilities
  - Achieves ~35 tokens per second on a GeForce 3060 with 12GB VRAM
- **Blockchain Integration**: Eliza supports agent wallets on any EVM chain with dozens of chains supported
- **State Management Issues**: Agents struggle to recognize external state changes (like wallet funding) without memory deletion

### Project Announcements
- **aigent.sh**: A new project built on ElizaOS technology aimed at making AI agents accessible to non-technical users
  - Releasing daily updates and offering prioritized early access
