## Model Instancing in Hyperfy V1 & V2
- **Hyperfy V1:** Instanced models still load textures separately, increasing file size and server load.
  - Solution: Use `alt+D` in Blender for linked duplicates and apply a script in `gltf.report` to instance the GLB.
- **Hyperfy V2:** Auto-instances linked duplicates from Blender, eliminating manual instancing.

## Texture Optimization for Low-Poly Builds
- Ensuring efficient texture usage when instancing models in Hyperfy V1.
- Solution: Use AI-generated image planes (e.g., Krea.ai) and project them onto low-poly extruded models.
