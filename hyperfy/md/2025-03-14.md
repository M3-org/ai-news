- Modified core to access and update vertices.

## Docker & Deployment Issues
- **0xpeezy** found Alpine base images break Hyperfy + Solana integration.
- Switching to `FROM node:22 AS build` resolved the issue.
- **hprivakos** noted auto-built Docker images lack git data, requiring a workaround.  
[Source](https://discord.com/channels/958209073277456457/994775534733115412)

## Image Processing for 3D Facades
- No need for upscaling; the process adds details around the subject.
- Useful for extruding 3D facades.
- Allows repainting images to match skybox colors for better blending.  
[Source](https://discord.com/channels/958209073277456457/958209074045026327)

## Non-Humanoid Rig Support
- Hyperfy does not support non-humanoid rigs.
- Workaround: Split the model into two objects in Blender and move the tail via script.
- Alternative: Use a base VRM armature and selectively use necessary bones.  
[Source](https://discord.com/channels/958209073277456457/1031058655581323324)
