# Hyperfy Development Updates – 2025-03-18  

## Technical Discussions & Decisions  

- **Mod Interoperability:** Mods modifying `createClientWorld/createServerWorld` cause conflicts. Solution: Convert all mods to the new plugin injection system.  
- **Namespace Conflicts in World Systems:** Using `world.solana` and `world.ethereum` namespaces was considered, but nesting makes safety harder.  
- **RigidBody Physics Enhancements:**  
  - New features: `rigidbody.addForce(force: Vector3)`, `rigidbody.addTorque(force: Vector3)`.  
  - Upcoming: `rigidbody.addForceAtPos(force, pos)`.  
- **MCP (Modular Code Plugin) Integration:**  
  - MCP added to the server, enabling real-time script updates via `hypkg`.  
  - Security improvements for remote connections are planned.  
- **Hyperfy-Link for IDE Integration:**  
  - Worlds can ping a local port to detect a Hyperfy-Link server and sync code with an IDE.  
  - Authentication for remote connections is needed.  
- **Hyperfy Documentation & Onboarding Issues:**  
  - Lack of clear messaging on Hyperfy’s purpose.  
  - Solution: Update homepage, GitHub README, and docs to clarify Hyperfy as a "meta game engine" with live collaboration.  
- **First-Person Mode in V2:** Works but may require manual setup.

## Three.js Integration & AI Development  

- **Three.js in Hyperfy:**  
  - Some Three.js primitives (e.g., `Vector3`) work.  
  - Full Three.js object support is not yet available.  
  - Users should use GLB models with added code instead.  
- **AI Integration in Hyperfy:** AI integration is in progress, with documentation and guides available.  
- **Hyperfy App & World Setup:** Worlds do not come with preinstalled apps; users must create them manually.

## UI & VR Compatibility Updates  

- **UI Padding/Margin Update:** Adjustments allow images to fit better within triangular pyramid shapes.  
- **Proximity Trigger Implementation:** A version with a proximity trigger was created but lacks a UI image.  
- **VR Compatibility:**  
  - The app runs on iPhone 12.  
  - Should work on newer Snapdragon XR2-based devices like Quest 3.  
- **Water Shader Update:** Implemented a scrolling diffuse and normal texture for water.
